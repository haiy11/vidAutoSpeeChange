# :tada: 智能变速插件 · 动态视频变速对比核心

> 基于画面变化幅度实时调整视频播放速度，让冗长片段自动快进，精彩瞬间慢放，重塑视频观看体验。

---

## ✨ 项目简介

你是否厌倦了手动拖拽进度条跳过视频中的平淡段落？  
**智能变速插件** 通过浏览器扩展与本地 Native 程序协同工作，实时分析视频帧画面的变化幅度，并 **动态调整播放速度** —— 画面变化剧烈时正常播放或慢放，画面长期静止或变化微小时自动快进。
适用于：
- 🎥 **监控录像** – 快速浏览无运动的时间段
- 🎬 **追番刷剧** – 对话剧情快速略过，专注画面变化大的精彩瞬间
- 🧪 **视频对比测试** – 精准控制变速区间，量化变化幅度

**工作流程**：
1. 扩展捕获视频当前帧（Canvas 截图）
2. 将帧图像通过 Native Messaging 发送给本地程序
3. 本地程序计算帧间差异 → 得到变化幅度值（0~100）
4. 扩展根据预设的速度映射规则，调整视频播放速度
5. 支持实时监听幅度曲线和监听画面播放，方便二次开发

---

## 🎮 核心功能

| 功能模块 | 描述 |
|---------|------|
| **智能变速引擎** | 基于画面像素差异度，动态计算目标播放速度 |
| **自定义速度区间** | 设置最低/最高速度，幅度阈值自由调节 |
| **多种变速预设** | 支持三种变速预设，能够一键切换 |
| **幅度监听面板** | 实时显示帧间变化波形，便于调参或验证 |
| **高性能本地计算** | C++ 实现，毫秒级响应，CPU/内存占用极低 |
| **跨浏览器支持** | 兼容 Edge / Chrome，遵循 Native Messaging 标准 |

## 🚀 快速开始

### 前置条件
- Windows 7 / 10 / 11
- Microsoft Edge (Chromium) 或 Google Chrome

### 安装步骤

1. **下载 Release**  
   从 [Releases](../../releases) 中下载项目并放置在电脑任意位置

2. **解压文件**  
   将 `vidAutoSpeeChangePage.zip` 文件解压到任意位置

3. **运行安装脚本**  
   在解压后的文件中进入 `vidAutoSpeeChangePage/intall`
   双击 `edge-install.bat` 或 `google-install.bat` 
   脚本将自动：
   - 生成 `com.dynamic.speed.json` 清单文件
   - 写入 Edge / Chrome 注册表项

5. **加载浏览器扩展**  
   - 打开 `edge://extensions` 或 `chrome://extensions`
   - 开启“开发者模式”
   - 点击“加载解压缩的扩展”，选择本项目的 `extension` 文件夹

###  开始使用  
   - 访问任何包含视频元素的网页，扩展图标将自动激活。  
   - 点击扩展突变，然后在弹窗中点击捕获按钮即可开始工作。
   - 在弹窗中点击设置选项，即可进入设置页面

### 卸载
- 运行 `uninstall.bat` 即可清理注册表项。其余文件便可放心删除
